# NEXUS AI Animated Website

A polished animated landing page for an AI robotics brand built with HTML, CSS, and JavaScript.

## Features
- Full-screen animated robot experience
- Smooth scroll-based frame transitions
- Responsive layout for desktop and mobile
- Ready to deploy as a static site on Vercel or any static host

## Run locally
Open the project folder and start a local server:

```bash
python -m http.server 8000
```

Then visit:

```text
http://127.0.0.1:8000
```

## Deploy on Vercel
1. Push this repository to GitHub.
2. Import the repo in Vercel.
3. Vercel will serve it as a static site automatically.
